Documents

 

 

How to use Documents?

Documents are the area where we can store the documents required by the solution in one place in the Solution folder.

Users can edit the documents from the Editor area.

New documents can either be created or imported in this section.

Go to the Resources tab and click on Documents.

 

To open the document in the File folder format, right-click and select the ‘Open Folder in File Explorer' option.

Users can paste the desired documents here.

 

Users can select the desired file from the folder.

 

Also, users can import files in the document by clicking on the Import Document.
