Introduction 

XML/JSON Tag Validation Action is used for validating XML or JSON tag in a file

 

 

How to Configure Action

Double click on the Business flow. Click on the ‘+’ icon

 

   On the Action Library screen go to the ‘General Actions’ tab and double click on the ‘File Transfer Action’

             

 

3) XML/JSON Tag Validation Action will be added under the Activity as shown below

 

Double click on it and go to the ‘Operation Settings’ tab.

The below screen should be displayed.

Select XML from dropdown against Document Type field

XML Template File Content can be given in the field.

            

 

Also, the user can select the check box against ‘Take Request from FilePath’

          

 

The Run Action will process the given set of data from the target file into the Processed File as per the XML Template given.

For JSON content/file follow the below steps:

Select JSON from dropdown against Document Type field

JSON Template File Content can be given in the field.

 

 

 

Also, the user can select the check box against ‘Take Request from FilePath’

 

The Run Action will process the given set of data from the target file into the Processed File as per the JSON  Template given.
