Introduction

This is used for validating the tags in the ‘XML or JSON’.

Pre-Conditions-Solution should be created with Agent like ‘Web', 'Mob’, or ’Power Builder’.

How to Configure Action

Double click on ‘Business flow’ then click on ‘Add New (+)’ button as shown in following screenshots.

 

Go to the ‘Actions Library’ tab which is highlighted below and then go to ‘General Actions’ tab and select the ‘XML/JSON Tag Validation Action’ then simply drag and drop into respective filed.

Below screen should be displayed.

XML Template File can be given using the Browse button. To use this, the ‘Take Request from file Path’ option must be checked.

The attributes can be added using the ‘Add New’ button as shown below.

You can also add ‘x-path’ manually in ‘Path’ field.

Click on save button to add the action.

 

The action gets added in Activity Action list

The Run Action button will validate the given Tags and it will display respective result.
