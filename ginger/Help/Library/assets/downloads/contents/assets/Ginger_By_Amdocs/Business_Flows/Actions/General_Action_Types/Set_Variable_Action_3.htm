Introduction

Set Variable is an Action that is used to create variables in the solution.

The below are the variable types: -

Dynamic

Selection List

Password String

Random String

Sequence

String

List

Random Number

Data Time

Timer

Variable List

How to configure this Action

Pre-requisite

A Solution should be created with web, mobile, or power builder agents.

           

           

Action Operation

Double click on ‘Business flow’ then click on ‘Variables’ and then click on the ‘Add New (+)’ button as shown below.

           

The ‘Add Variable’ window should be displayed. Select the desired variable type and

Click on the ‘Add Variable’ button.

Below ‘New Variable String’ screen should be displayed.

Enter the String value in the ‘Initial String Value'. You can also add a Description there.

You can also do ‘Set as input’, ‘Set as output’ and you can also publish it by clicking on publish option as shown in the screenshot.

Click on the Save button to add the Variable to the list.

            

The Variable gets added in Activity Variable list shown below.

      

This Variable can now be used in the Actions. Expand the ‘Activity Actions’ area

and click on the ‘Add New (+)’ button and then click on ‘Actions Library.’

The below screen is displayed. Go to ‘General Actions’ and select the ‘Set Variable Action’ and drag and drop into the respective field or click on that arrow to add as highlighted below.

The below screen is displayed.

The ‘Variable Name’ drop down displayed the list of variables that are added in the ‘Variables’ area.

‘Variable Type’ displayed the type of variable that is selected.

‘Set Variable Value Operation Type’ drop down displayed the action that is required to be done on the Variable. The options are ‘clear special

character’, ‘Set Value’, and ‘Reset Value.’

Click on the save button to save the action.

Click on the Save button to adds the Action to the Activity Actions
