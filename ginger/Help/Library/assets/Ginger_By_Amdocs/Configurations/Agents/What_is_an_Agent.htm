        What is an Agent? 

Agents are the drivers who are used for recording or execution for the target application under test.

 

       How to create an Agent?

When a solution is created, an agent specific to the platform is automatically added.

To add another agent, Go to Configurations Tab-> Agents. Click on the ‘+’ button to add a new agent as shown below.

      

 

 

In the below window, enter the Agent name. Select the Platform, Driver and relevant Type.

               

 

      Platform and Driver types: -

DOS – used for applications that are on DOC platforms.

                           It has a DOSConsole driver type.

JAVA – used for applications that are on Java platforms.

                            It has JavaDriver type.

MainFrame- used for applications that are on MainFrame.

                           It has a MainFrame3270 driver type.

Mobile - used for Mobile applications. It has an Appium driver type.

PowerBuilder - used for applications that are on Power Builder.

                          It has a PowerBuilder driver.

Unix- used for applications that are on Unix.

                           It has a UnixShell driver type.

Web- used for applications that are on a web platform. It has below driver types:-

SeleniumChrome

SeleniumFirefox

SeleniumIE

SeleniumRemoteWebDriver

SeleniumEdge

WebServices- used for applications that are using web services.

                           It has a WebServices driver.

Windows- used for applications that are on Windows.

                           It has WindowsAutomation driver type.

 

 

               How to map Web Agent to Business Flow?

             To use this Agent for the execution, we have to map the Agent to the Business Flows.

           When a new solution is created, the default Agent is already mapped to the Business Flow.

Mapping agent to the Business Flow

Go to the Business Flows tab and double-click on the required Business Flow.

Click on the ‘Change Agent Mapping’ button as shown below.

 

Click on the drop-down arrow then available agents get displayed. In the list only those agents will be displayed whose platform type is the same as the target application platform type.

Select the desired agent to map the agent.
