‎      How to Add Value Expression? 

The Value Expression can be added for an action using the ‘…’ button in the Edit Action window as shown below.

 

   

 

Click on ‘...’ button then below screen is displayed. It displays the variables (Global, Business Flow, Activities and Environments). Ginger provides the expression for Data Validation,Data Operations and Flow Details.

                    

 

Variables can be added to the expressions. To add desired variables, expand the Variables to see it and then double click on it to add it.

      

 

 

   The below screen displays a variable used in Date Time function. The button ‘Test Value Expression’ as shown below gives the

calculated Value of the expression after using the actual value of the variable in the ‘Value Calculated:’ field.
