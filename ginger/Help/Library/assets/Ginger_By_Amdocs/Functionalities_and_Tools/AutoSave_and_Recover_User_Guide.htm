  Purpose

            The user is working on Ginger for long hours and because of some unavoidable condition's Ginger crashes.

           Hence, we are providing the AutoSave and Recover mechanism to save user's recent work and give them an option to Recover their work.

   How does AutoSave work?

When the user will start working in Ginger, (at the start of an application), the AutoSave mechanism will be triggered.

After 5 minutes, every Business Flow and Run Set which you have worked on will be saved in the AutoSave folder. This will be created at the run time in your Solution folder path.

Run Set / Business Flow will be created/saved in the AutoSave folder as the same folder structure as the original one.

Once any Business Flow / Run Set has been saved, it will be removed from the AutoSave folder.

Only items that are unsaved and worked on will be saved in the AutoSave folder.

If you are closing Ginger manually then all the contents of the AutoSave folder will be cleared.

Right now, we are only storing Business Flow and Run Set files in the AutoSave folder.

The support for storing other files like Environment, Agents, etc. will be supported soon. In the coming Releases.

   How does Recover work?

If Ginger has been closed abruptly or has been crashed last time, then all the files from AutoSave folders will be moved to the Recover folder.

The recover folder will be created at the Solution folder path.

As soon as you will reopen Ginger, if there are any files in the Recover folder then a message will be shown on the UI with all the files listed inside it.

     

You can view details of each file, by clicking on the View Details Button or Double click event of a particular item.

  Delete Items

If the items/ work is not needed, you can delete it by clicking on the Delete Selected button.

  Recover Items

You can also recover the items which need to be recovered by clicking on Recover Selected button.

After deleting or recovering the items will be permanently removed from the list and the Recover folder.
