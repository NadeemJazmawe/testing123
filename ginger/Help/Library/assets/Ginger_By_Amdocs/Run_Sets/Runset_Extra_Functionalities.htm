How to Run Analyzer?

The Analyzer Run Set button is displayed as shown below in the drop-down.

 

Below Analyzer screen is displayed. It gives details of the Business Flow that has an issue along with the description of the issue,

the status if it can be auto fixed, and the severity of it.

 

   

 

If the ‘Can be Auto Fixed’ is Yes then click on the ‘Fix Selected’ button and the status changes to ‘Fixed’.

Click on the ‘Save Fixed Items’  button to save the changes in the Solution.

If the issue cannot be fixed automatically when the user needs to manually fix the issue before execution.

Users can also cross-check the Auto fix if it is as per the user expectation.

 

Click on the ‘Re-Analyze’ button to check the fix. The issue is not populated.

 

 

How to disable the Analyzer?

               The Analyzer can be disabled from the ‘Runners Execution Configurations’ option.

Uncheck the checkbox as shown below and execute the Run set. The Analyzer will not be called and the run set will be executed.

 

 

How to create Run set shortcut?

Click on the icon as shown below the ‘Create Run Set AutoRun Configuration’ option. 

 

 

The below screen should be opened.

 

  Click on the ‘Next’ button on the screen and keep the default selection

  Arguments: This option does not use any file for the execution configuration but only CLI arguments.

Dynamic: On run time from CLI, Ginger will load the existing or Virtual Run set, they will customize it according to the JSON configurations and then

will execute the customize Run set.

Click on the ‘Finish’ button and then User will get the below message

 

Go to the desktop and search for the shortcut as shown below.

  How to reload the Run set?

When a user works on the configurations of a Run sets, and he wants to restore the Run set to previously saved state i.e., he does not want to save his

new changes, then he can use the below button to reload the run set.

 

                Click on the ‘Yes’, the Run set will be loaded to the state where it was last saved.
