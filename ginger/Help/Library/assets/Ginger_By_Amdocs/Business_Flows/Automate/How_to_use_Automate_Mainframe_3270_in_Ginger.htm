Add Action

Set Text To Mainframe Action

  Locate By: Use By Caret Position or BY XY
 #x200e ByCaretPosition: Use the caret position shown in the Ginger mainframe window for the desired location.
‎ ByXY: We can also use the XY in the format x,y to set the location of the text
‎Set Text and Send Enter Checkbox: Tick this if you want to send enter after setting text.

 

Get Details from MainFrame Action
‎Similar to Set Text, Get Details also requires the Locate by and Locate Value.
‎
‎The value field is the length of the required text starting from the provided Caret Position/ XY.
‎If left blank it will return the text till the next whitespace.
‎

 

Send Keys to Mainframe

Use this Action to send Special Keys to the mainframe like entering, Tab Insert, Function Keys, etc.
‎ Users just need to select the desired key from the dropdown, no other configuration is required.
‎
