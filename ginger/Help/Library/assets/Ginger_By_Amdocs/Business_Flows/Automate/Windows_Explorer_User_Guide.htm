Description        

Windows Explorer allows the user to identify the elements and their properties.

Changes that were made in the solution will help the user to us Page Object Model integration for identifying elements on the page.

This is available for Web, Java, Windows and Power Builder.

Prerequisite

To activate the Explorer option, we have to activate the web agent.

Steps to use Ginger for Windows Explorer

  1. Select Windows Explorer

              Please follow the below steps for selecting windows explorer:

  1. Open Ginger
  2. Open Window Explorer from Automate Tab->Agent->Window Explorer (next to Agent on-Off button)
  3. Enter a webpage on the explorer.

                           

d. Click on the ‘Refresh’ button present on the top right of Window Explorer

                        

  1. Features

Following are the features and tabs that can be used to identify the elements on the page.

 

    1. Screenshot tab

 

  1. On this tab, we can select the element from the screenshot instead of moving to the browser window for selecting the element.
  2. Properties of each element and locating features can be validated at the same time.

          

  1. In Actions Tab – User has the option to choose elements on the page

                        

  1. Users can also integrate the page with existing POM.
  2. Select the “Integrated POM” checkbox for selecting POM models.

                             

 

Auto-Save and Auto-Update Check boxes to provide uninterrupted POM Elements updates and saving Page Objects Models after any update done from

explorer side without prompting user with alerts.

 

                            

    1. Page Source tab

This tab can be used to identify the source of the content.

    1. Tree View tab

This view will list down all the elements displayed on the page in form of a tree and could be useful to understand the structure of the page.

 

    1. Grid View tab

This view would help the user to select the element type and getting its path and other features in a grid format.

 

 

  1.           5.  Explorer Integrated Live Spy tab

Integrated Live-Spy tab introduced for ease of users to be able to identify elements via Live Spy.

Live Spy with new Element Details section to clearly display the relevant info regarding identified element with ability to test and add Action

(both regular as well as Integrated POM Action) directly via Action tab under Element Details section.
