What is a Variable? 

A Variable is an item that has a constant name and a value that can be changed.

How to create a variable?

Go to Automate Tab, expand the Variables area and click on the ‘+’ button to add a new variable as shown below.

Select the type of variable and click on the ‘Add Variable’ button.

Enter the Name and description.

A tag can be added using the ‘Add’ button which is used for grouping. (under development)

‘Value Expression’ is the area where a value is entered for the variable. Value can also be set using the Set Variable action.

‘Linked Variable’:- another variable of the same type can be linked so that one’s value is copied to another. Eg. the Business variable is linked to an Activity variable , hence whenever the value of the activity variable changes, the business variable gets updated.

‘Publish value’ will display the value from the linked variable.

Click on the ‘Generate Auto value’ button. This will populate the ‘Current value’ of the variable.

Click on the 'Ok' button to add the variable.

How to use a variable in an Action?

Variable can be used using the Set variable action

Click on the ‘Add New’ button in the Activity Actions section.

                 In the Add Action window, select the ‘Set Variable Action’ and Double click on the row to Add it to Action Tab.

Select the variable in the Variable Name dropdown. The variable type is automatically populated depending on the variable used.

Enter the value and click on the Ok button.

When this action is run, the variable value is set and is displayed in the current value field of the variable as shown below.
