Action Retry Mechanism

 

 

 

An Action can be re-executing multiple times if required.

This is used in situations like the web page is taking time to respond or when there is a connectivity issue.

All types of Actions have this facility of Retry Mechanism.

As shown below, the ‘Enable Retry Mechanism’ check box needs to be checked to use it.

The ‘Interval (seconds)’ is the waiting period before retrying to execute the Action.

The ‘Max Retries Number’ is the maximum number of times the action should be executed.
