Web API REST Action        

                                                                                 

                                    

                                                                         

Introduction

 

This action is used for Automating Rest API calls using Ginger

  

Pre-requisite

                         A Solution should be created with the Agent having a Web Services Platform

             

             

 

                         

Rest Action configuration

 

  1. Double click on ‘Business flow’ then click on the ‘Add New (+)’ button shown below and then click on ‘Actions Library’ shown below.

                                

                                 

  1. In the ‘Platform Actions’ tab, select the ‘WebAPI REST Action’ then simply drag and drop into the respective field or click on that arrow which is highlighted below.                                 

                            

  1. The below screen should be displayed. 
  • End point URL needs to be given.
  • The ‘Request Type’ has multiple options out of which the important one is ‘Get’ and ‘Post.’
  • ‘Response Content Type’ has 6 options like ‘Application/Json', 'Application/pdf’, ’Application/x-www-form-urlencoded',’Multipart/form-data',          'Text/plain: charset=utf-8',’xm’.
  • Cookies can be used if required.
  • Click on the Save button to add the action to the Activity Actions.

                 

  • The Request Body is mandatory for the ‘POST’ request type. Parameters can also be added here

                              

  • The Security Type can be added if required along with the Authorization details. In the Authorization drop down you will get different options.

                       

  1.       In order to preview the full raw request, press on the button “View Raw Request Preview”

                  

                   

                  The button will open a pop-up window with the Raw Request

 

                 

 

 

  1. Run Action will execute the API call
