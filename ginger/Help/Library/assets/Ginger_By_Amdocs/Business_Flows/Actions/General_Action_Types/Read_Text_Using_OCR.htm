Read Text using OCR Action

Ginger supports reading text from images or pdfs using OCR.  

OCR Stands for Optical character recognition and it is technology used for extracting the data from images, pdfs, or scanned documents.

 

Ginger action supports below operations

  • Read All Text from Image or PDF
  • Read text after Label from Image or PDF
  • Read text between two labels from Image or PDF
  • Read Text from Table in PDF

 

Add Read Text Action

To add Read Text Action to Ginger, go to Actions Library -> General Actions -> Read Text using OCR

                              

Select File Type

From the Read Text From dropdown, select your File Type as PDF or Image:
