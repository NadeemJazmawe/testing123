Introduction

This action is used to switch between windows.

How to Configure Action

Pre-requisite

                  A Solution should be created with Agents having Web, Windows, ASCF, Mobile, or Java Platform.

               

               

Action Operation

Double click on the Business flow. In the Activity Actions, click on the ‘Add New (+)’ button as shown below and then click on

‘Actions Library’ highlighted in the 2nd screenshot.

                

   

In the ‘Legacy Actions’ tab, select the ‘Switch Window Action’ then Simply drag and drop or click on that arrow which is highlighted in the

screenshot.

                  

You will get such a pop-up after selecting ‘Switch Window Action’. You can select yes and move forward.

Below screen should be displayed.

As it is part of the ‘UI Element Action’ name is ‘UI Element Action: Switch’

‘Locate By:’ drop-down can be used to indicate the locator.

The ‘Locate Value’ can be used using the ‘Open Window Explorer to select Element’ button.

Refer to the ‘Locate by-Locate Value’ document to know how to use this.

‘Sync Time (seconds)’ is used to enter the time required to Sync the windows.

Click on the save icon highlighted in the screenshot as shown below to add the action to the Activity Actions.

      Run Action will switch the focus to the given window in the Agent.
