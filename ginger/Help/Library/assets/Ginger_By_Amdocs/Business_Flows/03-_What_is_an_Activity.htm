What is an Activity?

 

 

An activity is an area in which a set of actions can be added. Each activity has a target application and actions specific to that target

application. However, application-agnostic actions can also be added.

 

  To add a new activity, do the following:

 

Click '+'.

The Add New Activity Wizard is displayed. On the Add New Activity Wizard, select an activity group from the drop-down menu. By default, the first activity group is selected

   

Click Next.

Enter the name, description, target application, and other information:

Click Finish.
